# Traffic-target-detection
<br>交通目标检测识别，包含：行人、人骑车、机动车<br>
<br>1.车辆：各种视角的作为交通运输工具的车辆，驾驶员有驾驶室;<br>
<br>2.行人：各种视角的行人，人推车或人准备骑车的状态算作行人;<br>
<br>3.人骑车：有人骑行的摩托车、电动车、自行车等<br>
<br>4.目标最长边≥80像素，且目标的遮挡不超过50%<br>
