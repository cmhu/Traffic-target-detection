# Traffic-target-detection
交通目标检测识别，包含：行人、人骑车、机动车
1.车辆：各种视角的作为交通运输工具的车辆，驾驶员有驾驶室
2.行人：各种视角的行人，人推车或人准备骑车的状态算作行人，蹲坐在地上或坐在轮椅上的行人不需要标注。
3.人骑车：有人骑行的摩托车、电动车、自行车等
